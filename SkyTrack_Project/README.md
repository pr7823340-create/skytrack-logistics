# SkyTrack Logistics - Dashboard de Drones

## Sobre o Projeto
Este é um protótipo de sistema logístico para gestão de drones,
desenvolvido para a disciplina de Desenvolvimento Web Front-End.

## Tecnologias Utilizadas
- HTML5 Semântico
- CSS3 (Flexbox e responsividade)
- Design Responsivo (Mobile First)
- Git & GitHub (Versionamento)

## Funcionalidades
- Visualização da frota de drones
- Cadastro de novos drones
- Exibição de alertas operacionais
- Layout adaptável para diferentes dispositivos

## Desafios Superados
- Implementação da tríade do truncamento (ellipsis)
- Validação de formulários com HTML5
- Criação de layout responsivo com Flexbox
- Organização de projeto com estrutura profissional
